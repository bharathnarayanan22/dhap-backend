## Decisions made (confirmed with Bharath)

1. **`User.id` is now `String`**, not UUID — matches `Task.id` and the new code;
   simplest for MongoDB.
2. **`inTask` added to `User`** — accept → `true`, task COMPLETED → bulk-cleared
   `false` for all assigned volunteers in a single `updateMany`.
